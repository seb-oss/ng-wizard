Get started with the wizard in four steps!

More information and examples for each step will be presented later on in this wizard.

Steps to get started

  1. Install - Install using npm with: npm install @sebgroup/ng-wizard --save
  2. Import module(s) - Import SebNgWizardModule and other dependencies.
  3. Add/use component - Add wizard component to app.
  4. Add steps (routes) - Declare steps used in the wizard.
